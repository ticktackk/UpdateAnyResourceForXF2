[HEADING=1]Update Any Resource for XenForo 2.1.10.2+[/HEADING]
[HEADING=1]Description[/HEADING]
Allows users with permission to add new update to any resource
[HEADING=1]Requirements[/HEADING]
[LIST]
[*]PHP 7.3+
[*]XenForo Resource Manager 2.1.4+
[*]Standard Library by Xon v1.10.0+
[/LIST]
[HEADING=1]Options[/HEADING]
[HEADING=3]XenForo Resource Manager[/HEADING]
[TABLE width="100%"]
[TR]
[th]Name[/th][th]Description[/th]
[/TR]
[TR]
[td]New resource update reply to discussion thread as resource creator[/td][td]If checked, post made in discussion thread about new resource update will be made as resource creator.[/td]
[/TR]
[/TABLE]
[HEADING=1]Permissions[/HEADING]
[HEADING=3]Resource moderator permissions[/HEADING]
[LIST]
[*]Update any resource
[/LIST]
