Update Any Resource for XenForo 2.0.6+
======================================

Description
-----------

Allows users with permission to add new update to any resource

Requirements
------------

- XenForo Resource Manager 2.0.0+
- PHP 5.6.0+

Permissions
-----------

#### Resource moderator permissions

- Update any resource
