Update Any Resource for XenForo 2.1.10.2+
=========================================

Description
-----------

Allows users with permission to add new update to any resource

Requirements
------------

- PHP 7.3+
- XenForo Resource Manager 2.1.4+

Permissions
-----------

#### Resource moderator permissions

- Update any resource
